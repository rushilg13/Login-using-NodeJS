# Login-using-NodeJS
# Used - <br>
Express <br>
Express-Session <br>
Mongoose <br>
Cookies <br>
2 Factor Auth <br>
Template Engine - Ejs <br>

# Routes
'/' - Register Page <br>
'/login' - User Login <br>
'/admin' - Admin Login <br>
'/admin_home' - Admin home page <br>
'/home' - User home page <br>
'/logout' - Logout <br>
'/reset' - reset password <br>
'/otp' - OTP page for password reset <br>
'/password' - Enter new password <br>

# How to run
Clone repo <br>
run npm install <br>
run node index.js <br>
http://localhost:4000/
